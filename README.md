# Examples in ESP8266 nodeMCU using FreeRTOS

  - Metronome
  - Display Tuner
  - Tuner with Metronome


### Prerequisites

Puts this examples into the directory esp-open-rtos/examples and into the respective paste do this:

```
$ make
$ make test
```
to run de example in nodeMCU.
